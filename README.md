## Acmo

「ActiveなMorning(朝活)を通して、1日をもっと有意義に過ごそう」というコンセプトをもとに開発した**朝活SNSサービス**

URL: https://acmo-app.com

## サービス概要

#### ターゲット(問題)

「朝活を継続したいけどモチベーションが続かない」

「1日のスタート時間が遅れてダラダラしてしまう」

といった問題を抱えたユーザー(自分です…)

#### メリット(解決策)
朝活内容の共有や仲間との交流によって、モチベーションの低下を防ぐ
